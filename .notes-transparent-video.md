# How to Create Transparent Desert Wind Video

## Export Settings (After Effects / Premiere / Similar)

1. **Format**: WebM
2. **Codec**: VP9 
3. **Enable Alpha Channel**: YES (this makes background transparent)
4. **Quality**: High
5. **Resolution**: Same as current (1920x1080 or similar)

## Alternative: PNG Sequence + FFmpeg

If your software doesn't support WebM with alpha:

```bash
# Export PNG sequence with transparency from your video editor
# Then use ffmpeg to create WebM:
ffmpeg -framerate 30 -i frame_%04d.png -c:v libvpx-vp9 -pix_fmt yuva420p desert-wind-transparent.webm
```
