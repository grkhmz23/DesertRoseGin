In both product sections on the Desert Rose Gin website there is an “Acquire” button. I want to replace those with a single reusable animated AcquireButton component that:

Lives in /components/ui/acquire-button.tsx

Uses Tailwind for styling

Uses framer-motion for subtle hover/tap animation

Uses lucide-react for the icon

Works with Typescript

Can accept standard button props and an optional label prop

Can be easily reused on both product sections

The visual style should feel premium and match a luxury spirits brand:

Rounded pill shape

Border and glow

Glassy / subtle gradient background

Slight hover lift and scale

Simple icon on the right side

New component code

Create this file at: /components/ui/acquire-button.tsx:

"use client";

import * as React from "react";
import { motion } from "framer-motion";
import { ShoppingBag } from "lucide-react";

export interface AcquireButtonProps
  extends React.ButtonHTMLAttributes<HTMLButtonElement> {
  label?: string;
}

export function AcquireButton({
  label = "Acquire",
  className = "",
  ...props
}: AcquireButtonProps) {
  const baseClass =
    "group relative inline-flex items-center gap-2 rounded-full border border-white/30 " +
    "bg-white/5 px-8 py-3 text-xs md:text-sm font-semibold tracking-[0.2em] uppercase " +
    "text-white/90 backdrop-blur-md transition-all duration-300 " +
    "hover:border-white/80 hover:bg-white/10 hover:text-white " +
    "shadow-[0_0_40px_rgba(255,255,255,0.18)]";

  return (
    <motion.button
      type={props.type ?? "button"}
      whileHover={{ y: -2, scale: 1.02 }}
      whileTap={{ scale: 0.97 }}
      className={`${baseClass} ${className}`}
      {...props}
    >
      {/* Highlight overlay */}
      <span className="pointer-events-none absolute inset-0 rounded-full bg-gradient-to-r from-white/15 via-white/0 to-white/20 opacity-0 transition-opacity duration-300 group-hover:opacity-100" />

      {/* Left glow dot */}
      <span className="pointer-events-none absolute -left-1.5 h-2 w-2 rounded-full bg-white/80 blur-[2px] group-hover:scale-110 group-hover:bg-white" />

      <span className="relative z-10">{label}</span>

      <ShoppingBag
        className="relative z-10 h-4 w-4 stroke-[1.6]"
        aria-hidden="true"
      />
    </motion.button>
  );
}

If the project does not have Tailwind, Typescript, or shadcn configured, do the following (adapt to existing tooling):

Typescript
If the project is not already Typescript, run the appropriate init command (for Next.js, Vite, etc.) and ensure .tsx components are supported.

Tailwind CSS
Install and configure Tailwind according to the official docs for the current stack, and ensure the Tailwind directives are present in the main CSS file:

@tailwind base;
@tailwind components;
@tailwind utilities;


shadcn
Initialize shadcn via CLI if not already done, and make sure reusable components live under /components/ui.

If the component root is currently something else, like /src/components, still create /components/ui and treat it as the main place for reusable UI primitives going forward.
